# 🎯 Cambridge B2 First (FCE) Success

Repository for tracking progress and storing study materials for the **Cambridge B2 First** certification.

## 📁 Repository Structure
* **/01_Use_of_English**: Key word transformations, phrasal verbs, and collocations.
* **/02_Writing**: Templates for Essays, Reports, and Reviews.
* **/03_Speaking**: Useful phrases for comparing photos and collaborative tasks.
* **/04_Vocabulary**: Daily word lists and common expressions.

## 🚀 Goals
- Master the **Use of English** structures.
- Improve fluency for the **Speaking** component.
- Build a solid **Vocabulary** base for the B2 level.
