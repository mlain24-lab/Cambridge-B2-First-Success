# B2 Essential Structures

| Concept (ES) | English Term | Explanation (??) |
| :--- | :--- | :--- |
| A menos que | **Unless** | Equivalent to "if not". Used in conditionals. |
| Acostumbrarse a | **To get used to (+ -ing)** | Describes the process of becoming familiar with something. |
| A pesar de | **In spite of / Despite** | Used to show contrast. "Despite" is never followed by "of". |
| Resulta que | **To turn out** | Used to describe an unexpected result or situation. |
| Por mi cuenta | **On my own / By myself** | To do something without help from others. |
