# Use of English - Quick Reference Tables

### Verbs + Prepositions

| Verb + Preposition | Meaning (ES) | ?? (Explanation / Rule) |
| :--- | :--- | :--- |
| **Depend + on** | Depende de | Used to express reliance: *It depends on the effort.* |
| **Co-operate + with** | Cooperar con | To work together: *Co-operate with your partner.* |
| **Succeed + in** | Tener éxito en | Followed by **-ing**: *He succeeded in passing.* |

### Linking Expressions

| Expression | Meaning (ES) | Example Sentence | Structure Rule (??) |
| :--- | :--- | :--- | :--- |
| **Owing to** | Debido a | *The delay was **owing to** the rain.* | Followed by a **Noun**. |
| **Whereas** | Mientras que | *I am tall, **whereas** my brother is short.* | Connects two contrasting clauses. |
| **Despite the fact that** | A pesar de que | ***Despite the fact that** it's late, I'll study.* | Followed by **Subject + Verb**. |
| **In order to** | Para / Con el fin de | *We study **in order to** get the B2.* | Followed by **Infinitive**. |
| **As long as** | Mientras / Siempre que | *You'll pass **as long as** you practice.* | Expresses a condition. |
| **Instead of** | En lugar de | *Use 'whereas' **instead of** 'but'.* | Followed by **Noun** or **-ing**. |
| **Despite + -ing** | A pesar de | ***Despite feeling** ill, he took the exam.* | **Never** use "of" with Despite. |
| **In spite of** | A pesar de | ***In spite of** the noise, I slept well.* | Always use **"of"** + Noun/-ing. |
| **Furthermore** | Además | *It's cheap. **Furthermore**, it's fast.* | Adds formal information. |
| **Nevertheless** | Sin embargo | *It was hard; **nevertheless**, we won.* | Formal contrast. |

## Refactoring Notes
* **Friendly:** Aunque termina en *-ly*, funciona como adjetivo (e.g., *User-friendly interface*).
* **Threaten:** Verbo esencial para reportes de incidentes (e.g., *The malware threatens the server integrity*).
