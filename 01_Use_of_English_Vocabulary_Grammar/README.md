## 5. Connectors & Advanced Grammar
Estructuras para conectar ideas complejas y condicionales técnicos.

### Connectors (Linking Expressions)
- **Despite / In spite of:** Seguido de Sustantivo o -ing (*Despite the low memory...*).
- **Despite the fact that:** Seguido de una frase completa (Sujeto + Verbo).
- **Nevertheless / However:** Contraste formal al inicio de una frase.
- **Consequently / Therefore:** Indica resultado o consecuencia.
- **In order to / So as to:** Indica propósito (seguido de infinitivo).
- **Owing to / Due to:** Indica causa (seguido de sustantivo).

### Advanced Structures
- **3rd Conditional:** `If + had + participle` -> *If I had had the snapshot, I would have restored the VM.*
- **Look forward to:** Siempre seguido de **-ing**. (*I look forward to starting the Master.*)
- **Suggest:** Seguido de **-ing**. (*The admin suggested updating the firewall.*)
- **Relative Pronouns:** **Whose** (Posesión), **Which** (Objetos/Sistemas).
- **Unless:** Indica condición negativa (*Unless we patch the system, it will be at risk*).

## 6. Common Errors (Fixes)
Correcciones basadas en errores comunes detectados en exámenes de Cambridge.

| Incorrect Syntax | Correct Syntax | Spanish |
| :--- | :--- | :--- |
| Agree *to* the opinion | **Agree with** | Estar de acuerdo con la opinión. |
| Concentrating *in* | **Concentrating on** | Concentrarse en algo. |
| Come *in* my country | **Come to** | Venir a mi país. |
| Fascinated *of* | **Fascinated by** | Fascinado por algo. |
| Taken *to* consideration | **Taken into** | Tener en cuenta / consideración. |
